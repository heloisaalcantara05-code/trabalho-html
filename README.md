CIRCUITO ZERO

Projeto de e-commerce desenvolvido com tecnologias web full stack para fins acadêmicos no primeiro período da graduação.

**Tecnologias**
- PHP
- JavaScript
- HTML5
- CSS3

**Funcionalidades**

- Cadastro e login de usuários
- Controle de sessão com PHP
- Carrinho de compras
- Simulação de pagamento via PIX, boleto e cartão
- Validação de formulários
